Scan the current branch for Claude/Anthropic/AI attribution that must be
removed before merge to master. Check commits, source files, and metadata.

## 1. Commit messages

Run `git log master..HEAD --format='%H %s%n%b'` (or against the branch point
if master doesn't exist locally). Flag any commit containing:
- `Co-Authored-By` (any casing)
- `Claude`, `Anthropic`, `AI assistant`, `AI-generated`, `LLM`
- `Generated by`, `Written by Claude`, `with assistance from`

For each hit, report the commit hash and the offending line.

## 2. Source files

Scan all tracked files in the diff range (`git diff --name-only master..HEAD`):
```
grep -rn -i 'co-authored-by\|anthropic\|claude ai\|generated by ai\|generated by claude' <files>
```

Exclude:
- CLAUDE.md, PLAN.md, .claude/ (these aren't committed)
- CHANGELOG entries that describe removing attribution (meta-references are OK)
- README mentions of tooling/methodology (if the project documents its process)

## 3. Git trailers

Check for git trailers that may have been added automatically:
```
git log master..HEAD --format='%(trailers:key=Co-Authored-By)'
```
Flag any non-empty results.

If hits are found, suggest the minimal fix:
- For commit messages: `git rebase -i` with specific parent hash

## Rules
- Read-only scan — do NOT modify commits or files
- Do NOT rewrite git history without explicit user instruction
- If the branch has been pushed, warn that rebase will require force-push
- Check `git log --format='%D' HEAD` to determine if remote tracking exists
